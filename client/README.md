## Client

### Requirements
* node v14 (or later)
### Getting started
1. Install deps
    ```sh
    npm i
    ```
2. Fill `.env to point to server
    ```
    cp .env.template .env
    ```
    then modify the newly created `.env`
3. Run client
    ```
    npm start
    ```
4. (Optional) Run tests
    ```
    npm run test
    ```
